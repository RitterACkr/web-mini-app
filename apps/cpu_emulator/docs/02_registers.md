# Registers (レジスタ)

`レジスタ` は CPU内部にある小さな記憶領域

CPUは計算を行う場合はメモリではなくレジスタを使用する

- CPUの内部に存在
- 非常に高速
- 数は少ない
- 一時的に値を保存する

---

## レジスタの役割

```
5 + 7 を計算するときCPUは
A = 5
B = 7
A = A + B
のようにレジスタを使って計算する
```
毎回メモリから読み書きする必要がなくなり,**計算が速くなる**

---

## このプロジェクト内のCPUで使用するレジスタ
- PC
- A
- B
- Z

### PC (Program Counter)

次に実行する命令の位置を保持する

```
PC = 0
なら, "次のメモリの0番地を実行する" ことを意味する
```

### Aレジスタ

主に計算に使用するレジスタ

### Bレジスタ

補助的な計算用レジスタ

### Zレジスタ

計算結果が0かどうかを記録する条件判定用のレジスタ

```
A = 0 のとき → Z = 1
それ以外は → Z = 0
```
