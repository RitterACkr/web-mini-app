# Branching (分岐)

`分岐` とはプログラムの実行位置を変更する事

通常, CPUは命令を順番に実行する
```
PC = 0
PC = 1
PC = 2
PC = 3
:
```
分岐を行う際は
```
PC = 0
PC = 1
PC = 2
PC = 10 <- ジャンプ
```
のように実行位置を変更できる

---

## 分岐の役割
- ループ
- 条件分岐 (if)
- プログラムの制御

が可能になる

---

## JMP

JMP は無条件で指定したアドレスにジャンプする

```
メモリ形式: [JMP][addr]

0x00: JMP
0x01: 0x05
-> PC=5にジャンプ
```

```
loop:
PRINTA
JMP loop

だと無限ループになる
```

---

## JZ (Jump if Zero)

Zフラグが1のときだけジャンプする

```
if Z == 1:
PC = addr

メモリ形式: [JZ][addr]
```
