# RAM + LOAD/STORE (メモリの読み書き)

`RAM` は実行中に CPU が読み書きするためのメモリ領域

ROM(プログラム領域)とは違い, RAMは値が変化する

このプロジェクト内ではメモリ `mem` を1つ持ち

- 低アドレス : プログラム (命令列)
- それ以外 : データ (RAM的に使用)

---

## LOAD / STORE
- LOAD : メモリの値をレジスタに読み込む
- STORE : レジスタの値をメモリに書き込む

### LDA_MEM

Aにメモリ値をロード
```
A = mem[addr]
Z = (A == 0)

メモリ形式: [LDA_MEM][addr]
```

### STA_MEM

Aをメモリにストア
```
mem[addr] = A

メモリ形式: [STA_MEM][addr]
```

---

## 実行例

addr=0x80 (アドレス値: 80) を変数として使うなら
```
1. A = 5
2. mem[0x80] = A <- STORE (Aの値をメモリに書きこみ)
3. A = mem[0x80] <- LOAD (書き込み直後のメモリからAにロード)
4. PRINTA
```
