# web-mini-app
小規模なWebAppの詰め合わせ

## Apps

| App | 説明 | 技術 |
|----|----|----|
| [ascii-rotate-cube](./apps/ascii-rotate-cube/) | ASCII文字で3Dキューブをレンダリング. 回転・ライティングをシミュレート | HTML / CSS / JS |
| [cpu-emulator](./apps/cpu_emulator/) | JSで実装した8bit最小CPUエミュレータ. レジスタ・ALU・メモリ・独自命令セットを持つ | HTML / CSS / JS |
| [othello-learning-ai](./apps/othello-learning-ai/) | ブラウザ上で強化学習するオセロAI. リアルタイムで学習させたAIと実際に対戦も可能 | HTML / CSS / JS |
| [web-calculator](./apps/web-calculator/) | ブラウザ上で動く電卓 | HTML / CSS / JS |

## 動かし方
各アプリフォルダに存在する `index.html` をブラウザで開くだけで動きます.
